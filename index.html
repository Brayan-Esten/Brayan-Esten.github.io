"Apa itu ARP?"

Address Resolution Protocol merupakan sebuah protokol dalam jaringan internet yang digunakan untuk mengkonversikan IP Address menjadi MAC Address untuk mengetahui alamat hardware. Di sini, kami akan menjelaskan simulasi protokol ARP dalam bentuk animasi yang bisa anda klik pada video di samping.

TERMS

Berikut penjelasan singkat mengenai beberapa terminologi yang terdapat dalam protokol ini.

1

IP Address

IP Address adalah sebuah alamat yang digunakan untuk mengidentifikasi perangkat dalam jaringan internet.

2

MAC Address

MAC Address adalah sebuah alamat fisik unik yang dimiliki tiap-tiap perangkat pada Network Interface Card.

3

ARP Cache

ARP Cache adalah sekumpulan IP Address beserta beserta MAC Address-nya yang dipetakan dalam bentuk tabel. Untuk tiap entry dalam cache ini ada dua jenis, Static Entry dan Dynamic Entry.

4

Static Entry

Static Entry adalah satu baris dalam ARP Cache berisi IP Address dan Mac Address-nya yang didaftarkan secara manual oleh kita sebagai end-user melalui command line. Static entry akan tetap berada dalam ARP Cache hingga kita memutuskan untuk menghapusnya. Hal ini dilakukan untuk mengurangi trafik jaringan karena dengan adanya static entry ini, kita tak perlu melakukan proses ARP lagi untuk mengetahui MAC Address tujuan.

5

Dynamic Entry

Dynamic Entry adalah satu baris dalam ARP Cache berisi IP Address dan Mac Address-nya yang didaftarkan secara otomatis ketika protokol ARP berhasil menresolusi IP Address menjadi MAC Address. Dynamic entry bersifat tidak permanen dan akan di-flush / dihapus secara berkala agar tidak memenuhi cache.
